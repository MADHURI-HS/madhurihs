# 🔗 URL Shortener

> **Production-ready URL shortening service built with Spring Boot**

### 📌 Overview

A scalable URL Shortener that converts long URLs into compact Base62 encoded links with persistent storage, Redis caching, API rate limiting, and Dockerized deployment.

---

### ⚡ Key Features

- 🔗 Base62 URL Encoding
- ⚡ Redis Caching
- 🗄 PostgreSQL Storage
- 🛡 Bucket4j Rate Limiting
- 🐳 Docker Support
- 🌐 RESTful APIs
- 📈 Click Tracking
- 🔄 Automatic Redirection

---

### 🛠 Tech Stack

| Category | Technologies |
|----------|--------------|
| Backend | Spring Boot |
| Database | PostgreSQL |
| Cache | Redis |
| Security | Bucket4j |
| Build Tool | Maven |
| Deployment | Render |
| Containerization | Docker |

---

### 🏗 Architecture

```text
               Client
                  │
                  ▼
          Spring Boot REST API
                  │
      ┌───────────┴───────────┐
      ▼                       ▼
   Redis Cache          PostgreSQL
      │                       │
      └───────────┬───────────┘
                  ▼
            Response
```

---

### 🎯 Highlights

- Reduced repeated database access using Redis cache.
- Implemented rate limiting to protect APIs.
- Dockerized application for easier deployment.
- Designed clean REST APIs following best practices.
